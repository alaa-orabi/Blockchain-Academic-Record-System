================================================================================
  REPORT CARD BLOCKCHAIN APP — COMPLETE RUNNING & TESTING GUIDE
================================================================================

────────────────────────────────────────────────────────────────────────────────
  PART 1 — FIRST-TIME SETUP (do this once)
────────────────────────────────────────────────────────────────────────────────

  STEP 1 — Activate your virtual environment (if you made one)
  ─────────────────────────────────────────────────────────────
  Windows (PowerShell):
      venv\Scripts\activate

  You should see (venv) appear at the start of your terminal line.
  Skip this step if you installed packages globally.


  STEP 2 — Install Python packages
  ──────────────────────────────────
      pip install web3 py-solc-x


  STEP 3 — Install Ganache (only once, needs Node.js)
  ─────────────────────────────────────────────────────
      npm install -g ganache


================================================================================
  PART 2 — EVERY TIME YOU WORK ON THE PROJECT
================================================================================

  Do these steps in order EVERY time you start a new session.


  STEP 1 — Open Terminal 1 and start Ganache (leave it running)
  ──────────────────────────────────────────────────────────────
      ganache --deterministic --accounts 10

  You will see 10 wallet addresses and "Listening on 127.0.0.1:8545".
  DO NOT close this terminal. It must stay open the whole time.


  STEP 2 — Open Terminal 2 and go to your project folder
  ────────────────────────────────────────────────────────
      cd path\to\your\project


  STEP 3 — Deploy the contracts
  ──────────────────────────────
      python scripts/1-deploy.py

  Expected output:

      Compiling GradeCoin.sol ...
      Deploying GradeCoin ...
        GradeCoin deployed at:   0x5b1869...
      Deployment complete!


  STEP 4 — Seed fake student data

  STEP 5 — Launch the terminal app
  ──────────────────────────────────
      python terminal_app/main.py

  You will see the banner, then a numbered list of 10 Ganache accounts.

================================================================================
  PART 4 — COMMITTING TO GITHUB UNDER YOUR NAME
================================================================================

  STEP 1 — Install Git (if not already installed)
  ─────────────────────────────────────────────────
  Download from: https://git-scm.com/download/win
  Run the installer with all defaults.
  Restart your terminal after installing.


  STEP 2 — Set your name and email (one time only)
  ──────────────────────────────────────────────────
      git config --global user.name  "Your Full Name"
      git config --global user.email "your@email.com"

  This is what appears on your commits on GitHub.


  STEP 3 — Create a .gitignore file in your project root
  ────────────────────────────────────────────────────────
  Create a file called  .gitignore  with this content:

      venv/
      __pycache__/
      *.pyc
      .env
      node_modules/

  This stops Git from uploading junk files.


  STEP 4 — Initialise Git in your project folder
  ────────────────────────────────────────────────
      cd path\to\your\project
      git init
      git add .
      git commit -m "Initial commit — Report Card blockchain app"


  STEP 5 — Create a repo on GitHub
  ──────────────────────────────────
  1. Go to https://github.com and sign in.
  2. Click the + button (top right) → New repository.
  3. Give it a name e.g. "report-card-blockchain"
  4. Leave it Public or Private (your choice).
  5. Do NOT tick "Add a README" (you already have files).
  6. Click Create repository.
  7. GitHub will show you a URL like:
       https://github.com/YourUsername/report-card-blockchain.git


  STEP 6 — Push your code to GitHub
  ───────────────────────────────────
      git remote add origin https://github.com/YourUsername/report-card-blockchain.git
      git branch -M main
      git push -u origin main

  GitHub may ask you to sign in — follow the browser pop-up.


  STEP 7 — Every time you make changes after that
  ─────────────────────────────────────────────────
      git add .
      git commit -m "Describe what you changed here"
      git push

  That's all. Your commits will appear under your name on GitHub.
